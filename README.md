# Fetch Random Fact Free API

This example demonstrates how to fetch data from a free public API using Python's standard library. It connects to a 'useless facts' API, retrieves a random fact, and prints it to the console. This illustrates a fundamental step in building side projects that leverage external services without cost.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
